# apagonapp.github.io
Official Website for ApagonApp

## What
Landing page for Apagón app using ASCII art style. Features app information, download links, and Puerto Rico power outage reporting details.

## Why
Create a clean, accessible website showcasing the Apagón app using the bokete.github.io terminal/ASCII aesthetic.

## How
- Copied bokete.github.io style (monospace, black/white, ASCII borders)
- Fetched content from apagonapp.com and web research
- Created index.html with app features, history, download links
- Downloaded official app icon
- Applied consistent terminal-style formatting
